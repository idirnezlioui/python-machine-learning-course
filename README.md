# Python Machine Learning - Cours et Travaux Pratiques

Bienvenue dans ce dépôt dédié à l'apprentissage du **Python appliqué au Machine Learning**.  
Ce cours est destiné aux étudiants souhaitant acquérir des bases solides en programmation Python avant d’aborder l’intelligence artificielle et l’analyse de données.

---

## Objectifs du cours

- Comprendre les fondamentaux du langage Python  
- Manipuler les structures de données (listes, dictionnaires, tuples, etc.)
- Créer et utiliser des fonctions et modules
- Découvrir les bibliothèques scientifiques : NumPy, Pandas, Matplotlib
- Appliquer les bases du Machine Learning supervisé et non supervisé
